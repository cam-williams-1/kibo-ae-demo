# kibo-ae-demo

A small shopping cart web application for Kibo. This is not exhaustive or fully polished, but it exemplifies the fundamentals of a web app using React components and API integration with clean, understandbale code.

## Technologies

- Vite 5.3 for Initial Build.
- JavaScript for Language.
- React 18.3 as Framework.
- Jest 30.2 and Babel 30.2 for Testing Suite.

### How To Run (Locally)

- Github Repository: (https://github.com/cam-williams-1/kibo-ae-demo)
- In the terminal, clone the repository to your computer then run npm run dev and the app will launch on your default browser.
- If it doesn't try these steps:
- Ensure localhost is set to 3000.
- Install npm.
- Check console for error messages or network errors.

### How To Test

- You can run npm test and the testing suite will execute in your terminal.
- You may need to install npm, jest, or babel - depending on your local machine.
